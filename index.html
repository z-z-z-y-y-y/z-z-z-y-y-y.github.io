高斯消元法详解（附洛谷 P3389 AC 代码）

使用场景

高斯消元法是一种经典的数值算法，主要用于求解线性方程组。当方程组规模适中且系数矩阵非奇异时，该方法能够高效地给出精确解。

核心思想

高斯消元法包含两个主要阶段：

  1. 消元阶段：通过初等行变换将增广矩阵转化为上三角矩阵。
  2. 回代阶段：自底向上求解，将矩阵进一步化为单位阵，得到方程组的解。

消元阶段的具体步骤

消元过程需要逐行推进。对于每一行（假设当前为第 i 行），都必须依次执行以下三个关键操作。

第一步：选取主元（列主元消去法）

在当前列（第 i 列）中，从第 i 行开始向下寻找绝对值最大的系数所在行，并将该行交换至第 i 行的位置。这一步骤能够有效避免因除数过小导致的数值不稳定问题。

第二步：判断主元是否为零

若经过换行处理后，当前的主元（即 determinant[i][i]）仍然为零，则说明系数矩阵是奇异的。此时线性方程组可能无解，或者存在无穷多组解。

第三步：归一化与消元

  • 归一化：将当前行的主元化为 1。具体做法是将该行所有的元素（包括常数列）都除以当前的主元值。
  • 消元：利用当前行，将位于该行之下的所有行中对应列（第 i 列）的元素消为 0。

完成上述循环后，原始的增广矩阵即被成功转化为上三角矩阵形式。

回代阶段

完成消元后，矩阵变为上三角形式。此时需要从最后一行开始，自底向上依次消去主元右侧的系数。经过回代，矩阵最终转化为单位阵，此时最后一列的数据即为方程组的解。

关键注意事项与易错点提醒

1. 关于主元选取的时机

这是本题最容易出错的地方。绝对不可以将选取主元与向下消元两个过程分离开来（即先一次性选好每一列的主元，再统一进行消元）。因为在高斯消元的进程中，矩阵的数值是动态变化的。如果在消元前就提前选定了主元，随着后续行变换的进行，最初选定的主元可能已不再是当前列的最优选择，这会直接导致计算结果错误。

正确做法：务必在每一行处理开始前，当场实时选取当前列的主元。

2. 关于精度处理

在判断主元是否为零时，需要考虑到浮点数运算的精度误差。

  • 在本题（洛谷 P3389）的数据范围下，使用 == 0.0 的判断方式通常已经足够。
  • 但在其他精度要求更高或数据范围更广的场景中，直接判断等于零可能会导致误判。建议改用容差比较的方式，例如：判断绝对值是否小于 1e-12 等极小阈值。

KMP 字符串匹配算法详解（附洛谷 P3375 AC 代码）

使用场景

在长字符串 s1 中查找给定字符串 s2 所有重复出现的位置（即字符串匹配问题）。

算法原理

KMP 算法通过预处理模式串 s2 的部分匹配表（前缀函数 π），利用模式串自身的重复模式，避免在匹配失败时回溯主串 s1 的指针，从而将匹配复杂度优化到 O(n + m)。

简单示例：
主串 s1 = "ABACABABC"，模式串 s2 = "ABA"。
注意到 s2 中前缀 "A" 与后缀 "A" 重复，因此当第一次匹配成功后，可以将模式串直接滑动到已匹配后缀的位置，避免对主串的重复检查。

复杂度分析

时间复杂度：O(|s1|+|s2|)
空间复杂度：O(|s2|)

核心步骤

  1. 构建前缀函数（π 数组）
    对模式串 s2 预处理，计算每个位置的最长相等前后缀长度。
  2. 字符串匹配
    遍历主串 s1，利用前缀函数跳过不必要的比较。

例题：洛谷 P3375 【模板】KMP

代码实现（C语言）

下面是我在洛谷上 AC 的代码，你也可以在 GitHub 上查看：p3375_KMP.c

#include <stdio.h>
#include <string.h>

void get_pi(char *s, int *pi)
{
    int len = strlen(s);
    int j = 0;
    pi[0] = 0;
    for (int i = 1; i < len; i++)
    {
        while (j > 0 && s[i] != s[j])
        {
            j = pi[j - 1];
        }
        if (s[j] == s[i])
        {
            j++;
        }
        pi[i] = j;
    }
}

void strcompare(char *s1, char *s2, int *pi)
{
    int len_s1 = strlen(s1);
    int len_s2 = strlen(s2);

    int j = 0;
    for (int i = 0; i < len_s1; i++)
    {
        while (j > 0 && s1[i] != s2[j])
        {
            j = pi[j - 1];
        }
        if (s1[i] == s2[j])
        {
            j++;
        }
        if (j == len_s2)
        {
            printf("%d\n", i - len_s2 + 2);
        }
    }
}

int main()
{
    char s1[1000005];
    char s2[1000005];
    scanf(" %s", s1);
    scanf(" %s", s2);

    int pi_2[1000005];
    get_pi(s2, pi_2);

    strcompare(s1, s2, pi_2);

    int len = strlen(s2);
    for (int i = 0; i < len; i++)
    {
        printf("%d ", pi_2[i]);
    }
    return 0;
}
